TNAH-Engine v1

The TNAH-Engine v1 was developed for Murdoch University unit ICT397 as a major project component for the unit that would be then used as a prerequisite for unit ICT398 the following semester.

TNAH stands for Team Names Are Hard which was our group name for the project for both units.

The engine was developed with some key requirements including: Runtime terrain generation, Basic enemy AI, Object/Model Loading and Animation and AI Object Avoidance. The project also required a playable game/demo which lead our team to create a clone of Doom named Zoom.

Built with C++ and OpenGL 3.3 leveraging libraries ASSIMP, GLM, GLFW and GLAD
